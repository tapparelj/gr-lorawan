Bitmask of configured regions.

Our input is a -D of one of CFG_eu868, CFG_us915, CFG_as923, CFG_au915, CFG_in866 More will be added in the the future. So at this point we create CFG_region with following values. These are in order of the sections in the manual. Not all of the below are supported yet.

CFG_as923jp is treated as a special case of CFG_as923, so it's not included in the below.

bitmask of` US-like regions

US-like regions have 64 fixed 125-kHz channels overlaid by 8 500-kHz channels. The channel frequencies can't be changed, but subsets of channels can be selected via masks.
